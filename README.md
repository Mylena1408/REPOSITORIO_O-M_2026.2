# MRC Projetos

Plataforma para Monitoramento e Suporte à Manutenção Preditiva da Cadeia Fria Industrial

## Sobre o Projeto

Sistema automatizado desenvolvido em n8n para monitorar equipamentos da cadeia fria industrial, identificar situações críticas de temperatura e gerar relatórios automáticos de apoio à manutenção preventiva e preditiva.
